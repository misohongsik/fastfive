Frontend Design & Interaction Debugging Checklist
Testsprite를 사용하지 않고, 프론트엔드 디자인 요소와 상호작용을 직접 검증하기 위한 체크리스트입니다.

1. Visual Effects (시각 효과)
Liquid Background (배경 애니메이션)
 PC: DeepDark 프리셋(검보라색) 흐름이 끊김 없이 부드러운가?
 Mobile: 픽셀이 깨지거나(계단 현상) 멈추지 않고 정상적으로 렌더링 되는가?
 에러 핸들링: Canvas 로드 실패 시 검은 배경(bg-black)이 정상적으로 노출되는가? (네트워크 차단 테스트 필요)
Glow & Text Effects (텍스트 효과)
 Hero H1: "FASTFIVE" 텍스트 뒤로 은은한 발광(Bloom) 효과가 보이는가?
 Hero Subtitle: "Premium Shared Office" 텍스트가 한 글자씩 타이핑되듯 나타나는가?
 Solution Title: "Everything Is Ready" 텍스트 위로 마우스를 올렸을 때, 스포트라이트(Flashlight) 효과가 마우스 커서를 따라오는가?
2. Global Navigation (네비게이션)
Navbar
 Sticky: 스크롤을 내렸을 때 상단에 고정된 상태로 따라오는가?
 Glassmorphism: 스크롤 시 배경이 반투명해지며 블러(Blur) 효과가 적용되는가?
 Logo Link: 로고 클릭 시 메인 페이지(/) 최상단으로 이동하는가?
 CTA Button: "Book a Tour" 버튼에 마우스를 올렸을 때 Glow 효과(빛)가 반응하는가?
3. UI Interactions (인터랙션)
Hero Section
 Book a Tour 버튼: 클릭 시 /tour 페이지로 즉시 이동하는가? (이전의 중첩 버튼 문제 해결 확인)
 Hover State: 버튼에 마우스를 올렸을 때 크기가 살짝 커지거나(Scale Up) 밝아지는가?
Tour Page
 Back to Home: 좌측 상단 링크가 배경과 구분되어 잘 보이는가?
 Link Action: "Back to Home" 클릭 시 메인 페이지로 돌아가는가?
 Form Input: 입력 필드 클릭 시 포커스 링(Focus Ring)이나 색상 변화가 있는가?
4. Mobile Responsiveness (모바일 반응형)
Layout
 Overflow: 가로 스크롤(좌우 흔들림)이 발생하지 않는가?
 Text Size: 모바일 화면에서 텍스트가 너무 작거나 잘리는 부분이 없는가? (Hero 섹션 H1 크기 확인)
 Touch Target: "Book a Tour", "Back to Home" 등의 버튼이 손가락으로 터치하기 충분한 크기인가?
Performance
 Scroll Jank: 모바일에서 스크롤 시 버벅거림(Jank) 없이 부드러운가? (Liquid Background 부하 확인)
 Initial Load: 페이지 진입 시 흰 화면(White screen)이 1초 이상 지속되지 않는가?
